# 🌱 OptiCrop

OptiCrop is an AI and Machine Learning based Smart Agriculture project that recommends the most suitable crop for cultivation based on soil nutrients and environmental conditions.

## 📌 Project Overview

The project uses Machine Learning algorithms to analyze agricultural parameters such as:
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Soil pH
- Rainfall

Based on these inputs, the system predicts the most suitable crop for farmers, helping improve productivity and decision-making.

## 🚀 How to Run

1. Clone the repository.
2. Install the required packages:

```
pip install -r requirements.txt
```

3. Run the application:

```
python app.py
```

4. Open the application in your browser.
